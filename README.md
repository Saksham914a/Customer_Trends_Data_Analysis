# 📊 Customer_Trends_Data_Analysis
Portfolio Project with end-to-end industry standard Data Analysis of Customer Shopping Trends from Retail Data using SQL, Python and Power BI.

Analyzed 3,900+ customer transactions using Python (Pandas), SQL, and Power BI to uncover insights into customer behavior, spending patterns, and product preferences. Performed data cleaning, feature engineering, and advanced SQL analysis to identify trends like high-value customers, top-rated products, and subscription impact. Built an interactive dashboard and provided data-driven business recommendations to enhance revenue and customer engagement.
